@font-face — Підключення власного шрифту

Завантажений нестандартний шрифт Dancing Script підключено двома способами: через @import з Google Fonts (у CSS) та продемонстровано синтаксис @font-face для локального файлу шрифту.

Синтаксис @font-face (для локального файлу): 
/* Завантажити з fontsquirrel.com або fontesk.com */
@font-face {
    font-family: "DancingScript";
    src: url('fonts/DancingScript-Regular.woff2') format('woff2'),
         url('fonts/DancingScript-Regular.woff')  format('woff'),
         url('fonts/DancingScript-Regular.ttf')   format('truetype');
    font-weight: 400;
    font-style: normal;
    font-display: swap;  /* уникаємо FOIT */
}

Google Fonts — Підключення через <link>

Підключено 3 шрифти: Playfair Display (заголовки, serif), Roboto (основний текст, sans-serif), Dancing Script (декоративний). Код підключення у <head>:

Підключення у HTML: 
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?
    family=Playfair+Display:ital,wght@0,400;0,700;0,900;1,400&
    family=Roboto:wght@300;400;500;700&
    family=Dancing+Script:wght@400;700&
    display=swap" rel="stylesheet">
Playfair Display

Веб-типографіка — мистецтво підбору шрифтів для цифрових носіїв. Serif-шрифт надає тексту класичного, академічного вигляду.

Найпопулярніший шрифт Google для веб та мобільних додатків. Чудово читається на екранах будь-якого розміру.

line-height

line-height: 1

Тісний міжрядковий інтервал. Текст важко читати при великій кількості рядків. Використовується рідко — лише для однорядкових заголовків.

line-height: 1.5

Стандартний міжрядковий інтервал для веб-тексту. Добре читається і є рекомендованим значенням для основного тексту сторінки.

line-height: 2

Широкий міжрядковий інтервал. Добре підходить для текстів з великим шрифтом або для статей з довгими абзацами.

  3. Уникати надмірної специфічності

Веб-формати шрифтів

  1. WOFF2 — найновіший, найлегший (рекомендований)
  2. WOFF — широко підтримується всіма браузерами
  3. TTF / OTF — класичні формати, більший розмір
  4. EOT — для IE (застарілий)
  5. SVG — векторний, для старих iOS (застарілий)

Маркований список із зображенням як маркером

Три варіанти: list-style-image: url(), власний маркер через ::marker, та background-image з відступом через padding-left.

Спосіб 1: list-style-image

  • Підключення шрифтів через @font-face
  • Google Fonts та іконкові шрифти
  • Font Awesome — популярний набір іконок
  • Властивості font-weight та font-size
  • Декоративне оформлення списків

Спосіб 2: background-image + padding-left

  • Адаптивний дизайн та медіа-запити
  • Flexbox та CSS Grid для верстки
  • CSS-анімації та переходи
  • Змінні CSS (custom properties)
  • Pseudo-elements ::before та ::after

Спосіб 3: Стилізація ::marker

  • JavaScript та DOM-маніпуляції
  • Робота з API та fetch-запити
  • LocalStorage та SessionStorage
  • Promises, async/await
  • Модульна система ES6+

Спосіб 4: Emoji-маркери через ::before

  • Вивчити основи HTML та CSS
  • Опанувати JavaScript та фреймворки
  • Практикуватись на реальних проєктах
  • Вивчити Git та командний рядок
  • Поповнювати портфоліо на GitHub
