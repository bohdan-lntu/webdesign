# Лабораторна робота №11. Контрольні питання

1. Функція — це іменований або анонімний блок коду, який можна багаторазово викликати для виконання дії або обчислення значення.
2. Функцію користувача створюють за допомогою `function name(parameters) { ... }`, функціонального виразу або стрілкової функції.
3. Формальні параметри — це імена в описі функції, фактичні параметри — це значення, передані під час виклику.
4. Функції без повернення значення виконують дію, а функції з `return` повертають результат у місце виклику.
5. Функцію можна записати у змінну, передавати як аргумент і викликати через цю змінну.
6. Функція є об'єктом, тому має властивості та методи, наприклад `name`, `length`, `toString()`.
7. `eval()` виконує рядок як JavaScript-код, але її не рекомендують використовувати без потреби через ризики безпеки й складність налагодження.
8. Глобальні змінні доступні в усьому скрипті, локальні існують лише всередині блоку або функції.
9. Значення параметра за замовчуванням задають у списку параметрів: `function f(x = 0) { ... }`.
10. Масивоподібний об'єкт `arguments` містить усі фактичні аргументи, передані у звичайну функцію.
