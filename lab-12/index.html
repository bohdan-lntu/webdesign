Теорія: Масиви JavaScript

Створення та доступ

let arr = [10, 20, 30];
arr[0]       // 10 (індекс з нуля)
arr.length   // 3
arr[arr.length - 1] // останній елемент

Додавання / видалення

arr.push(40)      // в кінець → [10,20,30,40]
arr.pop()         // з кінця  → повертає 40
arr.unshift(0)    // на початок → [0,10,20,30]
arr.shift()       // з початку → повертає 0
arr.splice(1, 1)  // видалити 1 ел. з індексу 1
arr.splice(1, 0, 15) // вставити 15 на індекс 1

Копіювання та об'єднання

arr.slice(1, 3)   // копія з 1 по 3 (не вкл.)
arr.concat([40, 50])  // об'єднання
[...arr, ...arr2]     // spread-оператор

Перебір та трансформація

arr.forEach(x => console.log(x));
arr.map(x => x * 2)       // новий масив
arr.filter(x => x > 10)   // відфільтрований
arr.find(x => x > 10)     // перший елемент
arr.reduce((sum, x) => sum + x, 0) // сума

Пошук

arr.includes(20)  // true / false
arr.indexOf(20)   // індекс або -1
arr.findIndex(x => x > 15) // індекс або -1

Сортування

arr.sort((a, b) => a - b)  // за зростанням
arr.sort((a, b) => b - a)  // за спаданням
arr.reverse()               // реверс масиву
