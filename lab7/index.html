Макетування з Flexbox та Grid

Сучасні CSS-технології дозволяють будувати гнучкі, адаптивні та зручні інтерфейси. Flexbox відмінно підходить для одновимірних макетів, а CSS Grid — для двовимірних сіток.

Одновимірна модель розкладки. Ідеальна для рядів і стовпців з гнучким розподілом простору.

Про лабораторну роботу

Дана лабораторна робота присвячена практичному застосуванню сучасних технологій CSS для створення адаптивних веб-макетів. Виконав студент групи КН-22 Барулін Богдан.

У роботі реалізовано: двоколонковий макет на Flexbox (sidebar + content), сітку карток на CSS Grid з адаптивними колонками, а також responsive-поведінку через media queries для мобільних, планшетних та десктопних екранів.
