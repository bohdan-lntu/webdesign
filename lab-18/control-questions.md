# Контрольні питання

1. Оптимізація сайту — це комплекс дій для покращення швидкості, SEO, зручності, адаптивності та ефективності вебресурсу.
2. Зображення оптимізують, щоб зменшити розмір сторінки, пришвидшити завантаження і покращити показники PageSpeed.
3. Для SEO використовують `title`, `meta description`, `meta keywords`, `meta author`, `robots`, а також правильну структуру заголовків `h1-h6`.
4. PageSpeed Insights показує швидкість і якість завантаження сторінки, зокрема LCP, INP, CLS, FCP, TTFB та рекомендації з оптимізації.
5. На швидкість сайту впливають розмір зображень, кількість CSS/JS, сторонні скрипти, кешування, хостинг, CDN, шрифти та структура HTML.
6. Основні види інтернет-реклами: контекстна реклама, банерна реклама, реклама в соціальних мережах, SEO-просування.
7. SEO дає органічний довгостроковий трафік через пошукові системи, а контекстна реклама дає швидкий платний трафік через рекламні оголошення.
8. Найефективніший вид реклами залежить від мети: для швидкого результату підходить контекстна реклама, для довгострокового просування — SEO, для точного таргетингу — соціальні мережі.
