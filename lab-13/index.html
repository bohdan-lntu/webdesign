Лабораторна робота №13

Тема: Робота з рядками у JavaScript.

Розробник: КН-22, Барулін Богдан.

Варіант: 2.

Теорія: Рядки JavaScript

Базові властивості

let s = 'Привіт, світ!';
s.length          // 13
s[0]              // 'П'
s.charAt(0)       // 'П'
s.toUpperCase()   // 'ПРИВІТ, СВІТ!'
s.toLowerCase()   // 'привіт, світ!'
s.trim()          // видаляє пробіли по краях

Пошук

s.indexOf('світ')      // 8 (або -1)
s.lastIndexOf('т')    // індекс останнього
s.includes('Привіт')  // true
s.startsWith('Пр')    // true
s.endsWith('!')       // true

Виділення підрядка

s.slice(0, 6)        // 'Привіт'
s.slice(-6)          // 'світ!'
s.substring(8, 13)   // 'світ!'

Заміна

s.replace('світ', 'JS')     // перший збіг
s.replaceAll('т', 'T')      // всі збіги
s.replace(/[аеиоуіє]/gi, '_') // регулярний вираз

Розбивка / з'єднання

'a,b,c'.split(',')    // ['a','b','c']
'abc'.split('')       // ['a','b','c']
['a','b'].join('-')   // 'a-b'

Шаблонний рядок

let name = 'Богдан';
`Привіт, ${name}!`     // вставка змінної
`${2 + 2}`             // вставка виразу
`Рядок
на два рядки`           // переноси рядка

Регулярні вирази

/pattern/flags         // літерал regexp
/\d+/.test('abc3')    // true — є цифра
'a1b2'.match(/\d/g)   // ['1','2']
flags: g=всі, i=без регістру, m=багаторядк.
