# Лабораторна робота №13. Контрольні питання

1. Текстову інформацію можна вивести через `document.write()`, `alert()`, `console.log()` або елементи DOM.
2. Конкатенація рядків виконується оператором `+` або шаблонними рядками з `${...}`.
3. Пошук у рядку виконують методи `indexOf()`, `lastIndexOf()`, `includes()`, `search()`.
4. Довжину рядка визначають властивістю `length`.
5. Підрядок виділяють методами `slice()`, `substring()` або `substr()`.
6. Заміну підрядка виконують методом `replace()` або `replaceAll()`.
7. Регулярні вирази використовують для пошуку, перевірки, виділення та заміни тексту за шаблоном.
8. Для роботи з регулярними виразами використовують `test()`, `exec()`, `match()`, `search()`, `replace()`, `split()`.
9. Підрядки за шаблоном можна отримати методом `match()` або через `exec()` у циклі.
