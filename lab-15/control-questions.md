# Лабораторне заняття №15. Контрольні питання

1. DOM — це об'єктна модель документа, яка представляє HTML-сторінку як дерево елементів.
2. `querySelector()` повертає перший елемент за CSS-селектором, а `querySelectorAll()` повертає всі знайдені елементи.
3. `addEventListener()` використовується для призначення обробника події елементу.
4. `onclick` у HTML не рекомендується, бо змішує структуру сторінки з логікою JavaScript і ускладнює підтримку коду.
5. Подія в JavaScript — це дія користувача або браузера, наприклад клік, введення тексту, натискання клавіші.
6. `event.preventDefault()` скасовує стандартну дію браузера, наприклад перезавантаження сторінки при відправленні форми.
7. HTML-елемент створюють методом `document.createElement("tag")`.
8. `classList` використовується для додавання, видалення, перевірки та перемикання CSS-класів.
9. `appendChild()` додає дочірній елемент у кінець вибраного батьківського елемента.
10. `textContent` працює з текстом як з безпечним текстовим вмістом, а `innerHTML` інтерпретує рядок як HTML-розмітку.
11. Список даних у JavaScript можна зберігати у масиві.
12. Масив об'єктів використовують, коли кожен елемент має кілька властивостей, наприклад `id`, `text`, `done`.
13. Фільтрацію елементів масиву реалізують методом `filter()`.
14. Модальне вікно через Escape закривають обробником `keydown`, який перевіряє `event.key === "Escape"`.
15. Сучасний JavaScript дозволяє працювати з DOM, подіями, класами та елементами без jQuery завдяки методам `querySelector`, `addEventListener`, `classList`, `createElement`.
