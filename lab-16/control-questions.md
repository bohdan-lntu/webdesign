# Лабораторна робота №16. Контрольні питання

1. Bootstrap — це фронтенд-фреймворк для швидкого створення адаптивних вебінтерфейсів.
2. До Bootstrap входять CSS-класи, адаптивна сітка, компоненти navbar, cards, buttons, forms, modal, dropdown, collapse та JS-компоненти.
3. Bootstrap дає готові класи й компоненти, а Vanilla CSS означає написання власних стилів без фреймворку.
4. Vanilla JavaScript — це чистий JavaScript без сторонніх бібліотек.
5. jQuery вважається застарілим для типових задач, бо сучасний JavaScript уже має `querySelector`, `addEventListener`, `classList` та інші зручні API.
6. Bootstrap реалізує адаптивність через grid system, контейнери, рядки й класи колонок для різних breakpoint-ів.
7. У CSS адаптивність реалізують через media queries, Flexbox, CSS Grid, відносні одиниці й responsive-обмеження.
8. Flexbox і Grid дають зручний контроль над розташуванням елементів, вирівнюванням, колонками, рядками й адаптивними макетами.
9. Більше контролю над дизайном дає Vanilla CSS/JS, бо всі стилі та поведінка створюються вручну.
10. Bootstrap доцільно використовувати для швидких прототипів, типових сайтів, адмінпанелей і проєктів, де потрібна готова адаптивна структура.
