Обрана CMS

WordPress — популярна безкоштовна CMS з відкритим кодом, великою кількістю тем, плагінів і зручною адміністративною панеллю.

Тип сайту

Інформаційний сайт студентської IT-спільноти.

Мова сайту

Українська.

Тема

Адаптивний шаблон у стилі освітнього блогу.

Створені сторінки

  • Головна — коротка презентація сайту.
  • Про спільноту — опис цілей і напрямів.
  • Медіа — галерея, відео-блок, календар і карта.

Хід роботи

  1. Ознайомлено з популярними CMS: WordPress, Joomla, Drupal, MODX, OpenCart.
  2. Для виконання обрано WordPress через простоту встановлення, теми та плагіни.
  3. Заплановано структуру сайту: головна сторінка, сторінка опису, медіа-сторінка.
  4. Підібрано тематичний шаблон для освітнього сайту українською мовою.
  5. Підготовлено текстове наповнення, галерею, блок відео, календар і карту.
  6. Оцінено адміністративну панель: сторінки, записи, медіафайли, меню, віджети та плагіни.

Оцінка CMS

WordPress зручний для швидкого створення сайту без глибокого програмування. Основні переваги: зрозумілий редактор сторінок, великий вибір тем, плагіни для галерей, форм, SEO й мультимедіа. Основні застереження: потрібно регулярно оновлювати ядро, теми й плагіни, стежити за безпекою та не встановлювати неперевірені розширення.
