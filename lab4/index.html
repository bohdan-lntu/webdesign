1. Базові селектори — Типографіка

Стилізація тегів h1–h3, p, списків та таблиці за допомогою селекторів тегів.

Заголовок третього рівня (h3)

Звичайний абзац тексту. CSS дозволяє керувати кольором, розміром шрифту, відступами та вирівнюванням елементів. Це перший абзац у секції.

Другий абзац у секції. Зверніть увагу на міжрядковий інтервал та відстань між абзацами — це задано через властивості line-height та margin.

Маркований список (ul)

  • Селектор тегу — p { color: red; }
  • Селектор класу — .card { padding: 16px; }
  • Селектор id — #main-title { text-align: center; }
  • Атрибутний селектор — [type="text"] { border: 1px solid; }
  • Псевдокласи — a:hover { color: blue; }

Нумерований список (ol)

  1. Відкрити редактор VS Code
  2. Створити файли index.html та style.css
  3. Підключити CSS через <link rel="stylesheet">
  4. Написати стилі та перевірити в браузері
  5. Валідувати код на validator.w3.org

Складні селектори (завдання 8)

  • .card.featured — картки з обома класами одночасно (рожева рамка)
  • .card:not(.featured) — картки БЕЗ класу .featured (сіра рамка)
  • .card:has(.description) — картки, що містять елемент .description (синя тінь)
  • .card:is(.featured, .new) — картки .featured АБО .new (зірочка в куті)
  • .card:where(.featured) — як :is(), але нульова специфічність

4. Форма — Атрибутні селектори та псевдокласи стану

Атрибутні селектори: [type="text"], [disabled], [required], [name^="user"], [name$="login"], [href^="https"], [href*="google"]. Псевдокласи: :focus, :valid, :invalid, :checked, :disabled.

5. Структурні псевдокласи

Демонстрація: :first-child, :last-child, :nth-child(), :nth-last-child(), :only-child, :nth-of-type().

:first-child / :last-child

  • Перший елемент (:first-child) — червоний
  • Другий елемент
  • Третій елемент
  • Четвертий елемент
  • П'ятий елемент (:last-child) — синій

:nth-child()

  • 1-й
  • 2-й — жовтий (:nth-child(2))
  • 3-й
  • 4-й — жовтий (:nth-child(2n))
  • 5-й
  • 6-й — жовтий (:nth-child(2n))

:nth-last-child()

  • 1-й з початку
  • 2-й з початку
  • 3-й з початку
  • 2-й з кінця (:nth-last-child(2)) — зелений
  • 1-й з кінця (:last-child)
