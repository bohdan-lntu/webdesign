## HTML

- Doctype HTML5 наявний.
- Вказано `lang="uk"`.
- Є `meta charset="utf-8"`.
- Є `meta viewport`.
- Є `title`, `description`, `keywords`, `author`, `robots`.
- На сторінці один заголовок `h1`.
- Використано семантичні елементи `header`, `nav`, `main`, `section`, `article`, `footer`, `address`.
- Для зображення задано `alt`.
- Email оформлено через `mailto`.
- Телефон оформлено через `tel`.
- Зовнішнє посилання має `target="_blank"` і `rel="noopener noreferrer"`.
- Незалежні HTML-блоки мають коментарі початку та кінця.

## CSS

- CSS винесено в окремий файл.
- Класи названо латиницею.
- Стилі згруповано за модулями.
- CSS-блоки мають коментарі.
- Для `body` вказано фон.
- Для `img` задано `max-width: 100%`.
- Використано Flexbox/Grid, а не таблиці.
- Є стани `:hover`, `:active`, `:focus`.
- Є media query для мобільних екранів.

## Функціональність

- Навігація веде до існуючих блоків.
- Логотип веде на головну сторінку.
- JavaScript винесено в окремий файл.
- Скрипт підключено через `defer`.
- Кнопка локальної перевірки працює через `addEventListener`.
