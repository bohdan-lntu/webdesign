HTML

Використано doctype HTML5, метатеги, семантичні елементи, один h1, атрибути alt і коректні посилання.

CSS

Класи названі латиницею, стилі згруповані за модулями, використано Flexbox, Grid і media queries.

Адаптивність

Макет перебудовується для мобільних екранів, картки переходять в одну колонку, навігація переноситься.

Результати перевірки

  • CSS винесено в окремий файл.
  • JavaScript винесено в окремий файл і підключено через defer.
  • Для зображення використано alt.
  • Зовнішнє посилання відкривається у новій вкладці.
  • Контакти мають посилання mailto та tel.
