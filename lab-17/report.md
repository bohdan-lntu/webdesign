# Лабораторне заняття №6 / Lab 17

**Тема:** Тестування та відлагодження коду.  
**Виконав:** КН-22, Барулін Богдан.  
**Варіант:** 2.

## Хід роботи

1. Створено HTML-сторінку з валідною структурою: `doctype`, `html lang`, `head`, метатеги, `title`, підключення CSS і JavaScript.
2. HTML-код приведено до чіткої ієрархічної структури з відступами.
3. Додано коментарі початку та закінчення незалежних блоків сторінки.
4. CSS-код винесено в окремий файл і згруповано за модулями: Base, Header, Hero, Sections, Footer, Responsive.
5. Додано адаптивність через media query для екранів до 780px.
6. Додано приклад зображення у форматі SVG з атрибутом `alt`.
7. Додано коректні посилання `mailto`, `tel`, зовнішнє посилання з `target="_blank"` та `rel`.
8. Додано JavaScript-перевірку базових елементів сторінки: `title`, `viewport`, один `h1`, `alt` у зображень.

## Перевірка

Локально перевірено структуру файлів, синтаксис JavaScript і наявність ключових вимог. Для повної зовнішньої перевірки сторінку можна вставити у W3C HTML Validator, CSS Validator, HTML Tree Validator і перевірити адаптивність через DevTools.

## Висновок

У роботі показано базовий аудит HTML і CSS коду, структурування сторінки, використання семантичних елементів, адаптивності та інструментів для локального відлагодження.
